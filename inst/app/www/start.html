This Shiny app implements quantitative bias analyses using episensr R package. Please report bugs at https://github.com/dhaine/apisensr/issues.

SIMPLE ANALYSIS TAB
For bias analyses requiring a 2-by-2 table as data input

SIMPLE ANALYSIS, NO OBSERVED DATA TAB
For bias analysis that do not have as input an observed 2-by-2 table.

PROBABILISTIC ANALYSIS TAB
For probabilistic bias analyses.
